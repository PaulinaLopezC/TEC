Simulador de Comunicación en C++ (Programación Orientada a Objetos TC1030)

Vivimos en un mundo donde las revoluciones tecnológicas ocurren todos los días.
Una de las revoluciones más importantes fue, sin lugar a dudas, los sistemas de
comunicación celular

¿Cómo crear un simulación de un sistema de comunicación celular?

Este proyecto esta diseñado para la interacción clientes, operadores y facturas.
El cliente cuenta con varias opciones como llamar, mandar mensajes o contar con
internet. Este programa nos permite visualizar la interacción de un cliente con
otro mediante llamadas y mensajes, la interacción de un cliente con los operadores
para consultar su factura total, la interacción de un cliente hacia el mismo, un
operador con otro cliente y casos similares. Este proyecto tiene el propósito de
implementar la programación orientada a objetos donde se utilicen los principios
de clases, atributos, funciones, herencia y polimorfismo con el objetivo de crear
un programa que nos permita la interacción de unos objetos con otros y que algunos
tengas la posibilidad de acceder a funciones de otras clases o atributos de otras clases.
